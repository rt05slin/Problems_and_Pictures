Problem #1 Is Your Team The Sum of a Bunch of Cubes?

It is fairly well known that the sum of the first N positive integers is N (N + 1) / 2. In other words, 1 + 2 + 3 + ... + (N-2) + (N-1) + N = N (N + 1 ) / 2. Less well known is that the sum of the first N positive perfect squares is N (N + 1) (2N + 1) / 6. In other words, 1 + 4 + 9 + 16 + ... + N2 = N (N + 1) (2N + 1) / 6. Even lesser well known is that the sum of the first N positive perfect cubes is (N (N + 1) / 2)2. In other words, 1 + 8 + 27 + 64 + ... + N3 = (N (N + 1) / 2)2.


INPUT- A positive integer N #< 1000, from file input1.

OUTPUT- The sum of the first N squares, to file output1.

EXAMPLE- Input: 5

Output: 55
