Problem #1 Fast Adder

There is a story of a grade school teacher who gave his class the problem of adding the first one hundred positive integers:

1 + 2 + 3 + ... + 98 + 99 + 100 =

A seven year-old boy immediately answered 5050 which is correct. The child prodigy Carl Friedrich Gauss developed into one of the greatest mathematicians of all time. Your computer program will be able to produce sums faster than Gauss could.

INPUT − Input from the keyboard a positive integer N < 32,767.

OUTPUT − Output to the screen the value N and the sum of the first N positive integers.

EXAMPLE − The following are examples with the input values in bold type.

Input: 5

Output: N = 5 SUM = 15

Input: 100

Output: N = 100 SUM = 5050
